# Badge

[Source code](https://github.com/chewhx/lyteui/blob/main/src/components/Badge/Badge.tsx)

```jsx
import { Badge } from 'lyteui';
```

## Overview

Present metadata or emphasize information with minimal word values. Usually used to indicate statuses of items in invoicing, boost, and cashouts.

	<Badge>Pending</Badge>

	<Badge theme="success">Success</Badge>
	<Badge theme="dark">Cancelled</Badge>
	<Badge theme="primary">Sent</Badge>
</Canvas>
