# Spinner

Render an animated spinner to indicate something is loading

```jsx
import { Spinner } from 'lyteui';
```

[View source code](https://github.com/chewhx/lyteui/blob/main/src/components/Spinner/Spinner.tsx)

| Props  | Type                 | Default   | Description             |
| ------ | -------------------- | --------- | ----------------------- |
| theme? | DefaultThemeColors   | "primary" | Change to a theme color |
| size?  | "sm" \| "md" \| "lg" | "mg"      | Adjusts overall size    |
