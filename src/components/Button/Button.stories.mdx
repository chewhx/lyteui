# Button

[Source code](https://github.com/chewhx/lyteui/blob/main/src/components/Button/Button.tsx)

```jsx
import { Button } from 'lyteui';
```

## Overview

Use buttons as interactive elements to let users trigger an action. They should be easy to locate among the UI.

<Canvas>
	<Button>Send invoice</Button>
</Canvas>

## API

<ArgsTable of={Button} />

## Loading

Present a loading button to indicate that the action has been successfully triggered

<Canvas>
	<Button isLoading>LytePay</Button>
	<Button isLoading variant="outline">
		LytePay
	</Button>
	<Button isLoading loaderPosition="right">
		LytePay
	</Button>
</Canvas>

## Button with icon

Provide some graphical context to the action handler

<Canvas>
	<Button icon={<FontAwesomeIcon icon={faUser} />}>Update profile</Button>
	<Button icon={<FontAwesomeIcon icon={faCircleCheck} />}>
		Accept invoice
	</Button>
</Canvas>

## Custom styles for loader and icon

Pass regular props to `loaderWrapperProps` or `iconWrapperProps` to style div containing loader and icon.

<Canvas>
	<Button
		icon={<FontAwesomeIcon icon={faUser} />}
		iconWrapperProps={{ style: { marginRight: '2rem', display: 'inline' } }}
	>
		LytePay
	</Button>
	<Button
		icon={<FontAwesomeIcon icon={faUser} />}
		isLoading
		spinnerWrapperProps={{
			style: { padding: '2rem', marginRight: '1rem', display: 'inline' },
		}}
	>
		LytePay
	</Button>
</Canvas>
